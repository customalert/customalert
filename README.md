
#  CustomAlert

It is a bewauiful custom tost alert like SweetAlert.


## Demo

https://customalert.github.io#examples


## Installation

Add This CDN To Your Site

Under Head Tag

```bash
<!-- CustomAlert JS CDN -->
<script src="https://cdn.jsdelivr.net/npm/@customalert/customalert@2.2.2"></script>
```
